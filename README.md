# time tracker - Chrome 拡張機能

Chrome ツールバーの拡張アイコンだけで操作する、8時間通知タイマーです。

---

## 現在の仕様

### 操作方法（ウィンドウ不要）

- 拡張アイコンを 1 回クリック: `0` で開始待ち（5秒）
- 5秒以内にもう 1 回クリック: `1`
- さらに 5秒以内クリックで `2`, `3` ... と増加
- 最大は `7`（以降は `7` のまま）
- 5秒間クリックが止まると、その数字で計測開始
- 停止したいとき: 拡張アイコンを右クリック → `ストップ`
- 音を確認したいとき: 拡張アイコンを右クリック → `音のテスト`

例:

- `2` で開始した場合、開始時点で 2 時間経過扱い
- そのため 8 時間通知までは残り 6 時間

### 通知仕様

- 8時間ごとに短く穏やかなチャイム音（約 0.8 秒・1回）を再生
- 通知後も停止せず、連続して 8 時間ごとに通知を継続
- 24時間表示ループの中で、8時間ごと（8,16,24...時間）に音が鳴る

### アイコン表示仕様

- 計測中の数値は 24 時間表示でループ
- 24時間ちょうどで `24` を表示し、その後 `0` に戻る
- 数字はできるだけ大きく表示されるよう自動調整
- 計測中の色は常に青（時間経過で色は変えない）

### 色

| 状態 | アイコン色 |
|------|------------|
| 未開始 | グレー |
| 数字選択中（5秒待ち） | オレンジ |
| 計測中 | 青 |
| 停止 | 赤（`STOP` 表示） |

停止表示は一定時間（約5秒）後に自動でデフォルト状態（グレー）へ戻ります。

## 手動インストール手順

1. Chrome を開き、アドレスバーに以下を入力して Enter

   ```
   chrome://extensions/
   ```

2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`time_tracker`）を選択
5. ツールバーに表示された拡張アイコンをピン留めして利用

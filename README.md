# time tracker – Chrome 拡張機能

8時間ごとに音で知らせる経過時間カウンター。

---

## 機能

| 状態 | ツールバーアイコン |
|------|-----------------|
| 未開始 | グレーの時計アイコン |
| 計測中（0〜8h） | **青**の円に経過時間の時数（0〜7） |
| 8時間超過 | **オレンジ**の円（アラーム音も鳴る） |
| 10時間超過 | **赤**の円 |

- **開始ボタン** → タイマー開始
- **やり直しボタン** → ゼロからリセット
- 8時間到達時に約4秒のビープ音で通知

---

## 手動インストール手順

1. Chromeを開き、アドレスバーに以下を入力して Enter：
   ```
   chrome://extensions/
   ```

2. 右上の **「デベロッパーモード」** トグルを **オン** にする。

3. 左上に現れる **「パッケージ化されていない拡張機能を読み込む」** をクリック。

4. このフォルダ（`time_tracker`）を選択して **「フォルダーの選択」**。

5. ツールバーにアイコンが現れたら完了！
   （表示されない場合はパズルピースアイコン → ピン留め）
